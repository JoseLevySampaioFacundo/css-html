TRABALHANDO COM LISTAS

Lista ordenada

  1. acordar
  2. tomar banho
  3. tomar o café de manhã
  4. escovar os dentes
    5.

o uso da tag<ol> faz uma lista ordenada juntamente com a atg <li>.

o uso do comando type= dentro do comando<ol>, permite com que o tipo de enumeração mude, mas essa enumeração muda apenas se no type começar com a primeira letra do alfabeto, primeiro número natural, etc.

Listas não ordenadas

  • arroz
  • carne
  • suco
  • caldo

o uso da tag<ul> permite fazer uma lista sem ordenamento. O mesmo vale no type e li sitados anteriormente.

No caso a seguir foi utilizado uma lista dentro de outra lista no caso dos elementos dentro das listas elas estão em ordenamento ou seja <ol>. Por outro lado, pode perceber que os elementos dentro de cada lista estão seguindo a ordem alfabetica das lestras, isso ocorre porque dentro da tag <ol> de ab ertura foi colocado o complemento start=, assim basta eu colocar apartir de qual elemento eu quero começar.
