Principais formatações

Negrito / Destaque

Nesta frase temos <b> para indicar uma frase em negrito (não semântico)

Nesta frase temos <strong> para indicar uma frase em negrito (semântico)

Itálico

Nesta frase temos <i> para indicar uma frase em Itálico (não semãntico)

Nesta frase temos <em> para indicar uma palavra em itálico (semântico)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

Texto grande e pequeno

estamos criando um texto grande e um texto pequeno nesse paragrafo

a tag <big>(absoleta) e a tag <small> são as tags utilizadas para essas modificações

Texto inserido / Texto deletado

o texto deletado é o texto que deve ser ligo, mas não considerado (uso da tag<del>)

o texto inserido é aquele em que a ma ênfase e indica que foi colocado depois (semântica) (uso da tag <ins>)

o texto inserido é aquele em que a ma ênfase e indica que foi colocado depois (não semântica) (uso da tag <u>)

Texto sobrescrito / Texto subscrito

textos sobrescrisro para colocar coisas do tipo X20+6 (uso da tag <sup>)

textos subscritos para colocar coisas do tipo H2O (uso da tag <sub>)
